PP片和芯板的型号

因为不同型号的PP片和芯板压出来的厚度不一样的，常用的PP片厚度如下

1080 厚度 0.075MM､

3313 厚度 0.09MM､

2116 厚度 0.115MM､

2116H 厚度 0.12MM､

7628 厚度 0.175MM､

7628H 厚度 0.18MM

因厂家而异，故我们需要找板厂要叠层的参数



jlc提供7628与3313



实际上，此选型主要是用于设计阻抗匹配

由于jlc不支持3313，使用经济型smt，所以选择7628

若不使用经济型smt，则尽量选择3313，加大板芯提高刚性



焊接时注意RGB位置，靠近按钮为R，左侧为G，最右侧为B
